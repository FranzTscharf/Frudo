# Frudo
Frudo, ein ToDo-System realisiert mit dem Django-Framework.
<br />
# Installation
`git clone https://github.com/phiphi282/Frudo` <br />
<br />
# Start
`cd todo/` <br />
`python manage.py runserver`<br />
